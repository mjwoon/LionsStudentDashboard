## 🎯 평가 점수 체계

### 점수 구성 (총 100점)

```
overall_score = 
    gpa_score                      × 25% +  # 전체 학점
    required_courses_score         × 30% +  # 진입요건 필수과목 ⭐ 최고 가중치
    recommended_completion_score   × 15% +  # 권장과목 이수 여부
    recommended_grade_score        × 15% +  # 권장과목 학점 수준
    curriculum_completion_score    × 15%    # 교육과정 완성도
```

### 등급 기준

| 점수 범위 | 등급 | 평가 |
|---------|------|------|
| 90-100 | A | 매우 적합 |
| 80-89 | B | 적합 |
| 70-79 | C | 보통 |
| 60-69 | D | 노력 필요 |
| 0-59 | F | 부적합 |

```
5개 과목 중:
  - Option A: A학점(4.0) 이상 1개
  - Option B: B학점(3.0) 이상 2개
```

**수강 과목:**
```
- GEN2052 (미분적분학1): A0 (4.0) ✅
- GEN2053 (미분적분학2): A0 (4.0) ✅
- GEN0063 (일반물리학1): B+ (3.5) ✅
- GEN0064 (일반물리학2): B0 (3.0) ✅
```

**평가 결과:**
```
required_courses_score = 100점
  ∵ Option A 충족 (A학점 2개 >= 1개)
  ∵ Option B 충족 (B학점 4개 >= 2개)
```

### Case 2: 진입요건 부분 충족

## 🧮 점수 계산 상세

### 1. GPA Score (25% 가중치)

```python
gpa_score = (current_gpa / max_gpa) * 100

예시:
- 강우수: (4.3 / 4.5) * 100 = 95.6점
- 강보통: (3.5 / 4.5) * 100 = 77.8점
- 강고민: (2.8 / 4.5) * 100 = 62.2점
```

### 2. Required Courses Score (30% 가중치) ⭐

```python
# 완전 충족
if 조건_만족:
    required_courses_score = 100

# 부분 충족
else:
    completed = 수강한_과목_수
    total = 전체_필수_과목_수
    required_courses_score = (completed / total) * 50
```

**예시 (전자공학부):**
```
진입요건: 5개 과목 중 A학점 1개 OR B학점 2개

강우수:
  - A학점 2개 → 조건 충족 → 100점

강보통:
  - B학점 2개 → 조건 충족 → 100점
  
강고민:
  - C학점 1개 → 조건 미충족
  - 수강: 1/5 → 10점
```

### 3. Recommended Completion Score (15% 가중치)

```python
recommended_completion_score = (수강한_권장과목 / 전체_권장과목) * 100

예시:
- 권장 20과목 중 17개 수강 → 85점
- 권장 20과목 중 12개 수강 → 60점
```

### 4. Recommended Grade Score (15% 가중치)

```python
recommended_grade_score = (권장과목_평균학점 / max_gpa) * 100

예시:
- 권장과목 평균 4.0 → 88.9점
- 권장과목 평균 3.2 → 71.1점
```

### 5. Curriculum Completion Score (15% 가중치)

```python
curriculum_completion_score = (이수한_전공체계도과목 / 학년별_권장과목) * 100

예시:
- 1-2학년 전공기초 10개 중 8개 → 80점
- 1-2학년 전공기초 10개 중 6개 → 60점
```

### 6. Overall Score (종합)

```python
overall_score = (
    gpa_score * 0.25 +
    required_courses_score * 0.30 +
    recommended_completion_score * 0.15 +
    recommended_grade_score * 0.15 +
    curriculum_completion_score * 0.15
)
```

**강우수 예시:**
```
overall_score = 
    95.6 * 0.25 +  # 23.9
    100.0 * 0.30 + # 30.0
    85.0 * 0.15 +  # 12.75
    90.0 * 0.15 +  # 13.5
    80.0 * 0.15    # 12.0
= 92.15점 → A등급 (매우 적합)
```

**강고민 예시:**
```
overall_score = 
    62.2 * 0.25 +  # 15.55
    20.0 * 0.30 +  # 6.0
    40.0 * 0.15 +  # 6.0
    50.0 * 0.15 +  # 7.5
    35.0 * 0.15    # 5.25
= 40.3점 → F등급 (부적합)
```
