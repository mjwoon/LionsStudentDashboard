# 🚀 관리자 기능 빠른 테스트

## 1단계: 서버 실행

### 백엔드 서버
```bash
cd backend
uv run fastapi dev main.py
```

✅ 서버가 http://localhost:8000 에서 실행되는지 확인

### 프론트엔드 서버  
```bash
cd frontend
npm run dev
```

✅ 서버가 http://localhost:5173 에서 실행되는지 확인

## 2단계: 관리자 페이지 접속

1. 브라우저에서 http://localhost:5173 접속
2. 상단 네비게이션에서 **"관리자"** 클릭
3. "데이터 업로드" 탭이 표시되는지 확인

## 3단계: 샘플 파일 다운로드

각 섹션의 **"샘플"** 버튼 클릭:
- 과목 데이터 → `sample_courses.json` 다운로드
- 학생 데이터 → `sample_students.json` 다운로드  
- 수강 데이터 → `sample_enrollments.json` 다운로드

## 4단계: 데이터 업로드 테스트

### 과목 데이터 업로드
1. "파일 선택" 클릭
2. `sample_courses.json` 선택
3. "업로드" 버튼 클릭
4. 결과 확인:
   ```
   ✅ 과목 데이터 업로드 완료
   ✓ 새로 추가: 1개
   ✓ 업데이트: 0개
   ```

### 학생 데이터 업로드
- 위와 동일

### 수강 데이터 업로드
- 위와 동일

## 5단계: 백엔드 API 직접 확인 (선택)

Swagger UI로 확인:
```
http://localhost:8000/docs
```

"admin" 태그 섹션에서:
- POST `/api/admin/upload/courses/file`
- POST `/api/admin/upload/students/file`
- POST `/api/admin/upload/enrollments/file`

## 문제 해결

### 1. "서버 연결 실패"
→ 백엔드 서버가 실행 중인지 확인

### 2. "CORS 오류"
→ 백엔드 서버 재시작

### 3. "파일 형식 오류"
→ JSON 형식 확인 (샘플 파일 참고)

### 4. 아무 반응 없음
→ F12 → Console 탭에서 오류 확인
